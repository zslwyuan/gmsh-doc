434  * MQuadrangle
435  *
436  * 3--3+3E-...--4+2E--2
437  * | | E = order - 1;
438  * | | N = total number of vertices
439  * 4+3E 3+2E
440  * | | Interior vertex numbers
441  * ... 4+4E to N-1 ... for edge 0 <= i <= 3: 4+i*E to 3+(i+1)*E
442  * | | in face : 4+4*E to N-1
443  * 3+4E 4+E
444  * | |
445  * | |
446  * 0---4--...---3+E---1
447  *
